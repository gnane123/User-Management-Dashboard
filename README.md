 # User Management Dashboard

## Setup

npm install

## Run

npm run dev

## API

https://jsonplaceholder.typicode.com/users

## Features

- View Users
- Add User
- Edit User
- Delete User
- Search
- Sort
- Filter
- Pagination
- Responsive Design
- Form Validation
- Error Handling

## Assumptions

- JSONPlaceholder is used as mock backend.
- Department field is simulated because API does not provide one.

## Challenges

- JSONPlaceholder does not persist CRUD operations.
- Mapping API data to custom UI fields.
- Managing filters and pagination together.

## Future Improvements

- Redux Toolkit
- Server-side pagination
- Infinite scrolling
- Dark mode
- Unit testing with Jest
- Material UI
